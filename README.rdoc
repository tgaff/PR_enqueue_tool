# Poll github to enqueue jenkins builds

## Usage
### configuration
* Works through environment variables mostly.  You can use a shell script to save the variables and load them between builds:

```
export GITHUB_USER=tom
export GITHUB_TOKEN=29c0d9fe03079dfdf1bd8e3ca2d2d0a6f3e5f423
export GITHUB_REPO=pudding
export JENKINS_JOB=name_of_job_we_want_to_run_on_jenkins
```

### Running
* rake enqueue:run
